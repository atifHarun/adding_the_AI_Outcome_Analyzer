Build a clean, modern web platform prototype for an AI Governance Marketplace and AI Trust Layer.

The current system already includes a Credit Score Fairness Monitoring tool that accepts a CSV dataset and calculates fairness metrics. This tool already exists and should remain functional.

The goal is to wrap the existing tool inside a multi-step AI governance platform demo that is intuitive and impressive for investors.

The platform should simulate a Managed Service Platform (MSP) where organizations can evaluate, monitor, and certify their AI systems.

Design the UI so it feels modern, minimal, and enterprise-grade, similar to products like Stripe, Vercel, or Notion dashboards.

Use clear navigation, cards, charts, and step-based progress.

IMPORTANT:
Do not rebuild the existing fairness monitoring logic. Only integrate it into the new multi-step platform UI.
